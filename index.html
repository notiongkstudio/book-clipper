🔗 Connect to Notion

Step 1: Go to notion.so/profile/integrations → Click the "Internal integrations" tab → "+ New integration" → Name it "Book Clipper", select your workspace, skip everything else → Click Save → Copy the Internal Integration Secret (click Show, then Copy).
Step 2: Open 📚 My Library database as a full page → Click ⋯ (top right) → Scroll down and click "+ Add Connections" → Search "Book Clipper" → Confirm.
Step 3: Paste your token and database ID below. You only need to do this once — it saves automatically.

💡 To find your Database ID: open My Library as a full page → click ⋯ → Copy link → paste it anywhere → the 32-character string before ?v= is the ID.
